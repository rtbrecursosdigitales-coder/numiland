# 🌟 NumiLand - Tu Aventura Matemática

**NumiLand** es una plataforma educativa gamificada diseñada para que niños de todas las edades aprendan y practiquen matemáticas de una manera divertida, visual y emocionante.

## 🚀 Características Principales

*   **🏆 Sistema de Progresión**: Niveles desbloqueables con recolección de estrellas y medallas.
*   **🌍 Mundos Inteligentes**: Contenido adaptado por edad:
    *   **Exploradores (5-7 años)**: Conteo visual y comparación básica.
    *   **Aventureros (8-10 años)**: Operaciones fundamentales y secuencias.
    *   **Maestros (11-13 años)**: Álgebra temprana y geometría.
    *   **Leyendas (Secundaria)**: Retos avanzados.
*   **🎙️ Accesibilidad**: Lectura de voz automática para los desafíos (ideal para niños que están aprendiendo a leer).
*   **👤 Perfiles Personalizados**: Elige tu nombre y un avatar único entre más de 20 opciones.
*   **☁️ Sincronización en la Nube**: Gracias a Firebase, el progreso se guarda automáticamente al iniciar sesión con Google.
*   **✨ Interfaz Animada**: Experiencia fluida con efectos visuales modernos y divertidos.

---

## 🛠️ Tecnologías Utilizadas

*   **Frontend**: React 18 + Vite (Máxima velocidad de carga).
*   **Estilos**: Tailwind CSS (Diseño responsivo y moderno).
*   **Animaciones**: Framer Motion (Transiciones suaves).
*   **Base de Datos y Auth**: Firebase (Firestore para el progreso y Google Auth).
*   **Iconos**: Lucide React.
*   **Voz**: Web Speech API para síntesis de voz en español.

## ⚙️ Instalación y Configuración

Si deseas ejecutar este proyecto localmente:

1.  **Clona el repositorio**:
    ```bash
    git clone https://github.com/tu-usuario/numiland.git
    ```
2.  **Instala las dependencias**:
    ```bash
    npm install
    ```
3.  **Configura Firebase**:
    Crea un archivo `.env` o configura las variables de entorno con tus credenciales de Firebase.
4.  **Corre el servidor de desarrollo**:
    ```bash
    npm run dev
    ```

## 🎨 Personalización

Puedes modificar los siguientes archivos para ajustar la experiencia:
*   `/src/App.tsx`: Cambia el link de pago de Mercado Pago o ajusta la lógica de niveles.
*   `/src/lib/levels.ts`: Añade o modifica los problemas matemáticos y los niveles.
*   `/src/index.css`: Cambia los colores de la marca en la sección de temas de Tailwind.
